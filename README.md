## Deploy in 2 minutes (GitHub Pages)

```bash
cd ~/Desktop/portfolio
git init
git add .
git commit -m "Portfolio launch"
git remote add origin https://github.com/prakhyathMyGIT/portfolio.git
git push -u origin main
```

Then in GitHub → Settings → Pages → Source: `main branch / root` → Save.

Your portfolio is live at: `https://prakhyathMyGIT.github.io/portfolio`
